## 🚀 Getting Started

### Requirements

- Node.js 20+
- npm
- Firebase Project
- Gemini API Key

### Installation

```bash
git clone https://github.com/Royalbulls/RBA-AI-Pro---Legal-Business-Document-Engine.git

cd RBA-AI-Pro---Legal-Business-Document-Engine

npm install
```

Create

```
.env.local
```

```env
GEMINI_API_KEY=YOUR_API_KEY

FIREBASE_API_KEY=

FIREBASE_AUTH_DOMAIN=

FIREBASE_PROJECT_ID=

FIREBASE_STORAGE_BUCKET=

FIREBASE_APP_ID=
```

Run

```bash
npm run dev
```

Production Build

```bash
npm run build
```
